# matrix_rain
C implementation using Ncurses.

Use program argument to change character color. Like "./matrix_rain red".
Available colors:
  green as default,
  red,
  yellow,
  blue,
  magenta,
  cyan,
  while.
  
Available buttons:
  Enter - skip intro,
  Key up - increased speed rain,
  Key down - reduction speed rain,
  q - exit.
